### [View the Demo](https://github.com/undefinedhuman/sts-blog)

## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Installation

> This guideline assumes you already have Node.js and npm installed.

```sh
git clone https://github.com/undefinedhuman/sts-blog.git
cd sts-blog
npm install
```

## Usage

```sh
npm run start
```

## Testing

```sh
# Run all test suites
npm run test
```
